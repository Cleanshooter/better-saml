SAML 2.0 Single Sign-On allows you to use a SAML 2.0-compliant Identity Provider for Single Sign-On to your blog.

== Description ==

Better SAML is based on the SAML 2.0 Single Sign-On plugin by ktbartholomew however includes several improvements, primarily allowing WordPress to identify between SAML and non-SAML users and allow them to bypass the system automatically rather than manually.
